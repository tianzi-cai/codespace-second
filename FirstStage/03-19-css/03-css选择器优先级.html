如何判断选择器的优先级

1. 是否是行内样式 2. ID选择器数量相加 3. 类选择器、属性选择器、伪类选择器的数量 4. 标签选择器、伪元素选择器的数量 5. 看顺序 例如： 1. p#abc.test.test2[title] 2. #abc#abc1.test[title]:hover A:是否是行内样式，是为1，不是为0 A B C D 1. 0 1 3 1 2. 0 2 3 0 例如： 3. p#abc.test.test2[title] 4. #abc.test[title]:hover A:是否是行内样式，是为1，不是为0 A B C D 3. 0 1 3 1 4. 0 1 3 0
